# pūra.baby — Organic Baby Skincare

A D2C organic baby skincare brand website targeting Indian parents. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks or build tools required.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Landing page — hero, product grid, certifications, testimonials |
| `product.html` | Product detail page — gallery, variants, ingredient decoder, reviews |
| `404.html` | Custom error page |

## Features

- Fully responsive (mobile, tablet, desktop)
- Animated hero with floating product cards
- Interactive ingredient decoder with expandable safety ratings
- Size & variant selectors with live price update
- Pincode-based delivery date check
- Sticky add-to-cart bar on scroll
- Tab-based content (How to use / Ingredients / Reviews)
- Zero external JS dependencies

## Local development

Just open `index.html` in your browser — no build step needed.

```bash
# Optional: use a local server to avoid CORS issues with fonts
npx serve .
```

## Deployment

Hosted on GitHub Pages at:

See the full deployment guide for setup instructions.
